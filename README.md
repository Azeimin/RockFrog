# RockFrog

Проект заморожен. 

Данный репозиторий используется для разработки школьного проекта - социально-игровой сети.

Используемые языки программирования:

- Python
- Javascript

Основные используемые технологии при разработке проекта:

- Django Framework
- Django channels
- Redis
- Websocket
- Ajax
- sqlite
- nginx

Зависимости(pip3):

django 3.1.8
django-model-utils
channels 2.4.0
channels-redis
