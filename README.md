# RockFrog

Данный репозиторий используется для разработки школьного проекта - социально-игровой сети.

Используемые языки программирования:

- Python
- Javascript

Используемые технологии при разработке проекта:

- Django Framework
- Django channels
- Redis
- Websocket
- Ajax

Зависимости(pip3):

django
channels
channels-redis
pillow
